# Todo App API

A simple Todo API built with FastAPI.
Users can register, log in, and manage their todos.

## Features

* User Register
* User Login
* Create Todo
* Get Todos
* Update Todo
* Delete Todo

## Tech Stack

* FastAPI
* Python
* JWT Authentication
* SQLite

## Run the Project

```bash
git clone https://github.com/itxmuhammadwaqarali/todo_app.git
cd todo_app
pip install -r requirements.txt
uvicorn main:app --reload
```
